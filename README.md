# Игра от кота Василия

Ролевая игра с пошаговой боевой системой, реализованная на Python в рамках учебного задания.

## Модули

### Модуль 1
- Абстрактный класс `Unit` с шестью базовыми характеристиками (сила, ловкость, телосложение, мудрость, интеллект, харизма).
- Три абстрактных метода: `calculate_max_health()`, `calculate_damage()`, `calculate_defense()`.

### Модуль 2
- Класс `Character`, наследующий `Unit`.
- Атрибут `character_class` (warrior / mage / hunter).
- Реализация методов расчёта здоровья, урона и защиты с использованием ветвления по классу.
- Формулы соответствуют требованиям (округление вниз).

### Модуль 3
- Абстрактный класс `Spell` с полями `name`, `damage`, `mana_cost` и методом `cast()`.
- Конкретные заклинания: `Fireball`, `IceLance`, `LightningBolt`.
- В класс `Unit` добавлены атрибуты `spells` (список) и `mana` (текущая мана), методы `add_spell()` и `cast_spell()`.
- В класс `Character` добавлен метод `calculate_max_mana()` с формулами для каждого класса.
- При создании персонажа мана устанавливается равной максимальной.
- Невозможно создать экземпляры абстрактных классов `Unit` и `Spell`.

## Пример использования

```python
from game_classes import Character, Fireball, IceLance

# Создаём мага
mage = Character(10, 12, 8, 18, 20, 15, 'mage')
print(f"Мана: {mage.mana}/{mage.max_mana}")

# Добавляем заклинания
mage.add_spell(Fireball("Огненный шар", 35, 15))
mage.add_spell(IceLance("Ледяное копьё", 25, 10))

# Применяем заклинание
damage = mage.cast_spell(0)
print(f"Урон от заклинания: {damage}, осталось маны: {mage.mana}")
